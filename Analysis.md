# Sales Data Analysis Report

## Objective

Analyze sales data using Python and pandas.

## Dataset

The dataset contains:

- Product Name
- Quantity Sold
- Price

## Data Cleaning

- Missing Quantity values replaced with 0
- Missing Price values replaced with average price
- Duplicate rows removed

## Metrics Calculated

- Total Revenue
- Total Quantity Sold
- Average Sale
- Best Selling Product
- Highest Revenue Product

## Findings

- Laptop generated the highest revenue.
- Mouse sold the highest number of units.
- Missing values were handled successfully.

## Tools Used

- Python
- pandas
- VS Code
